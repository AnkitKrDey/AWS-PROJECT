About This Project

This project showcases how to host a static website using AWS S3. It involves creating an S3 bucket, uploading HTML content, and setting permissions for public access. This is a part of a hands-on DevOps learning experience.

Key Features

  • ✅ Hosted on AWS S3
  • ✅ Public Access Enabled
  • ✅ Lightweight and Fast
  • ✅ Fully Static Content
